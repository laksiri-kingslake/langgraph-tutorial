# LangGraph Tutorial

## Configure Development Environment
1. Create and activate virtual environment
```bash
# create virtual environment
python -m venv myenv

# activate
source myenv/bin/activate
```

2. Install requirements if any
```bash
pip install -r requirements.txt
```
